# How to use

## Overview

This project shows boilerplate code for integrating your solana wallet into a website with a simple interface.
It is designed for educational purposes for those who want to understand by trying both the program code and the interface,
with a simple interface running on localhost, to show how to interact with the wallet by performing a few simple
functional operations after connecting to the Solana wallet.

## Installation

1. Open the project folder in your preferred IDE such as Visual Studio Code (VS Code).
2. Install dependencies using:
   - If using yarn, run the command `yarn`.
   - If using npm, run the command `npm install`.

## Usage

1. After successfully installing dependencies, run the development server by executing:
   - `yarn dev` if you are using yarn.
   - `npm run dev` if you prefer npm.
2. Once the server is running, open your web browser and navigate to the localhost address provided by the server.
